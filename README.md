# data-analysis-portfolio
Репозиторий содержит проекты, демонстрирующие мои навыки и стэк задач по аналитике данных.

- **Языки:** SQL (PostgreSQL)
- **Визуализация:** Metabase
- **Инструменты:** Excel, Git

## 📂 Проекты

| Проект | Описание | Основной стек |
|--------|----------|---------------|
| [RFM-сегментация](./RFM/) | Кластеризация клиентов на основе поведения. Построение дашборда в Metabase и формирование рекомендаций. | SQL, Metabase |
| [ABC/XYZ-анализ](./ABC-XYZ/) | Многшомерный анализ по трем признакам в связке с анализом спроса. | Excel |
| [Юнит-экономика](./Юнит-экономика/) | Расчет юнит-экономики продукта: CAC, LTV, ARPPU, маржинальность. | Excel |

## 🚀 Как использовать
1. Перейдите в папку интересующего проекта
2. Откройте его README для детальной инструкции
3. Следуйте шагам по воспроизведению анализа
